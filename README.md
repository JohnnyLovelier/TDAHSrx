# DIVA 2.0 — Entretien diagnostique interactif pour le TDAH chez l'adulte

Outil interactif basé sur l'entretien DIVA 2.0 (Kooij & Francken, 2010 — DIVA Foundation).

## Fonctionnalités

- 18 critères DSM-IV avec exemples cochables (âge adulte + enfance)
- Évaluation du retentissement dans 5 domaines de vie
- Critères B, C, D et E du DSM-IV
- Score automatique et diagnostic (type combiné / inattentif / hyperactif-impulsif)
- Sauvegarde automatique dans le navigateur (localStorage)

## Déploiement

```bash
npm install
npm run dev
```

Déploiement sur Vercel : connecter le repo GitHub, Vercel détecte automatiquement Next.js.

## Avertissement

Cet outil est une aide à la passation et ne remplace pas le jugement clinique d'un professionnel de santé qualifié.
